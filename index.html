Alt Text Validator

Suggested Format: Subject, action, context. A _____ does something from somewhere.

Alt text must inclue:

Proper punctuation and complete sentences must be included in alt text.

Use third person. First and Second preson should only be used if writing text that appears in the image or using the embodiment technique to instruct user to do something or imagine something.
